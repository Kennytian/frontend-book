# HTML meta
很多前端开发对 HTML 的`<meta>`标签视而不见，经常会忽略它的存在，认为它没有太大的作用，现在我们就谈谈`<meta>`这个标签的使用。

## meta 标签的作用

* 搜索引擎（SEO）优化；
* 定义页面使用语言；
* 自动刷新页面；
* 控制页面缓存；
* 网页定级评价；
* 控制页面显示的窗口；
* 等等...

## meta 使用
`<meta>`标签总共有 3 个属性，不同的属性和值组成了网页不同的功能：

* name
* http-equiv
* content

### name 属性
`name`属性主要是用于描述网页的，对应`content`属性中的内容是便于搜索引擎查找和分类信息。
